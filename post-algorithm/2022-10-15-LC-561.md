# 문제

Given an integer array nums of 2n integers, group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized. Return the maximized sum.

Example 1

```md
Input: nums = [1,4,3,2]
Output: 4
Explanation: All possible pairings (ignoring the ordering of elements) are:

1. (1, 4), (2, 3) -> min(1, 4) + min(2, 3) = 1 + 2 = 3
2. (1, 3), (2, 4) -> min(1, 3) + min(2, 4) = 1 + 2 = 3
3. (1, 2), (3, 4) -> min(1, 2) + min(3, 4) = 1 + 3 = 4
   So the maximum possible sum is 4.
```

Example 2

```md
Input: nums = [6,2,6,5,1,2]
Output: 9
Explanation: The optimal pairing is (2, 1), (2, 5), (6, 6). min(2, 1) + min(2, 5) + min(6, 6) = 1 + 2 + 6 = 9.
```

Constraints:

1 <= n <= 104
nums.length == 2 \* n
-104 <= nums[i] <= 104

# 풀이

문제조건상 2n으로 항상 페어가 될 것이고 정렬을 했을때 각 자 두개로 쪼갠것의 앞에 것이 항상 작은값이 되는 것이다.
인덱스는 하나 작게 시작하니까 0부터 +2를 해주며 계속 더해주면 원하는 값이 완성된다.

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
/*
 [6,,6,5,] -> [1,2,2,5,5,6] => 1,2, 2,5, 6  6 이 베스트 페어고
 [1,2,3,4] -> 12 34 가 베스트 페어네
1. 정렬을 함
2. 길이는 2n이 될 거임
정렬을 한 이상 min값은 두개씩 페어했을때 앞에것이 될 것임
걔네들 다 더하면 되는데
*/
var arrayPairSum = function (nums) {
  nums = nums.sort((a, b) => a - b);
  let answer = 0;

  for (let i = 0; i < nums.length; i += 2) {
    answer += nums[i];
  }
  return answer;
};
```
